Smart Parking System

GitHub Link

The **MAT Smart Car Parking System** is an automated solution designed to streamline parking management using sensors, microcontrollers, and real-time monitoring. By detecting vehicle presence through IR or ultrasonic sensors, the system guides drivers to available slots via LED indicators or a mobile app, reducing search time and congestion. It integrates barrier controls, digital payment options, and centralized tracking to optimize space utilization, enhance security, and minimize human intervention. Ideal for smart cities, malls, and corporate campuses, this system improves efficiency, reduces emissions, and ensures a hassle-free parking experience.

Car Game

GitHub Link

"SpeedRacer Kotlin" is a fast-paced 2D car game built using Kotlin, where players control a high-speed vehicle dodging obstacles, collecting power-ups, and racing against the clock. With intuitive touch/swipe controls, dynamic difficulty levels, and vibrant retro-style graphics, the game offers an addictive arcade experience. Features include real-time scoring, fuel management, and unlockable cars, all optimized for smooth performance on Android devices. Built entirely in Kotlin, this game showcases clean code, object-oriented design, and efficient collision detection for a seamless mobile gaming experience
